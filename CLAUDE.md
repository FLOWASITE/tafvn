## Architecture

### Routing (`src/routes/`)
File-based routing via TanStack Router. File name ‚Üí URL:
- `index.tsx` ‚Üí `/`
- `dich-vu.tsx` ‚Üí `/dich-vu` (services index)
- `dich-vu.$slug.tsx` ‚Üí `/dich-vu/:slug` (dynamic, `$param` not `[param]`)
- `__root.tsx` ‚Üí wraps every route; contains `<Header>`, `<Footer>`, `<Chatbot>`
- `sitemap[.]xml.ts` ‚Üí `/sitemap.xml` (server route, no React)

**Never manually edit `src/routeTree.gen.ts`** ‚Äî it's auto-generated by TanStack Router on dev/build.

### Server Functions (`src/lib/*.functions.ts`)
TanStack Start `createServerFn()` ‚Äî RPC-style calls that run server-side only. All use the admin Supabase client (bypasses RLS). Inputs validated with Zod. Examples: `getPage(slug)`, `listPagesByType(type)`, `listOffices()`.

### Supabase (`src/integrations/supabase/`)
- `client.server.ts` ‚Äî admin client with service role key; **server-side only**
- `client.ts` ‚Äî browser client (publishable key)
- `types.ts` ‚Äî auto-generated TypeScript DB types; regenerate with Supabase CLI after schema changes

### Static Config (`src/lib/site.ts`)
Single source of truth for static data: `SITE`, `NAV`, `SERVICES` (6 service types), `PROVINCES` (9 regions), `FAQ_GENERAL`. Always update here rather than hard-coding strings in components.

### Components
- `src/components/ui/` ‚Äî shadcn/ui primitives (Radix UI + Tailwind). Add new ones with `bunx shadcn@latest add <component>`.
- `src/components/site/` ‚Äî custom branded components. Consume `ui/` components and apply TAF's design language.

## Database Schema (Supabase)

| Table | Purpose |
|-------|---------|
| `pages` | Headless CMS ‚Äî `slug` (PK), `type` (service/province/article/static), `title`, `h1`, `body_html`, `og_image`, `author_id`, `faq` (JSONB), `published_at`, `noindex` |
| `authors` | Content authors ‚Äî `name`, `title`, `credentials`, `bio`, `avatar_url` |
| `offices` | Office locations ‚Äî address, coords, `is_primary`, `sort_order` |
| `contact_leads` | Contact form submissions |
| `redirects` | `from_path` ‚Üí `to_path` with 301/302 status |

RLS policies make `pages` publicly readable only when `published_at IS NOT NULL AND published_at <= now()`.

## Key Conventions

**File suffixes matter:**
- `.server.ts` ‚Äî server-only; never bundled to client
- `.functions.ts` ‚Äî contains `createServerFn` calls; also server-only

**Styling:** Tailwind CSS 4 with CSS variables. Base color: slate. Fonts: Playfair Display (headings), Source Serif 4 (body), Inter (UI) ‚Äî all with Vietnamese subsets. shadcn/ui style: `new-york`.

**Content language:** All copy is Vietnamese. No i18n framework ‚Äî content is statically typed/fetched.

**Forms:** `react-hook-form` + Zod resolver pattern throughout.

**Path alias:** `@/*` resolves to `src/*` (configured in `tsconfig.json` and Vite).
